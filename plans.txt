1.0 DB struktūras izzinašana
1.0.1. lietotaja vards - SELECT CURRENT_USER();
1.0.2. aktīva DB - SELECT DATABASE();
1.0.3. lietotāja tiesibu noskaidrošana - SHOW GRANTS FOR 'user'@'localhost'
1.0.4. visas DB - SHOW DATABASES;
1.0.5. DB izvele - USE DBNAME; ( var parbaudit ar komandu no 1.0.2)
1.0.6. DB veidošana - CREATE DBNAME;
1.0.7. DB dzēšana - DROP DATABASE;
1.0.8. visas tabulas - SHOW TABLES;
1.0.9. tabulas struktūras apraksts - DESCRIBE TABLE_NAME; (SHOW COLUMS FROM TABLE_NAME;)
1.0.10. tabulas detalizets apraksts un statistika - NAME TABLE STATUS FROM DBNAME;
1.0.11. tabulas ierakstu izgušana - SELECT * FROM TABLE_NAME;
1.0.12 tabulas ierakstu izgusana ar filtru - SELECT * FROM TABLE_NAME WHERE COLUMN_NAME=VALUE;
* filtrs var būt modificēts(daudzu limeņu)
1.0.13.tabulas veidošana - CREATE TABLE TABLE_NAME(COLUMN_NAME_DATATYPE...);
... - var būt vairākas īpašības:
	NULL/NOT NULL
	DEFAULT
	PRIMARY KEY
	AUTO_INCREMENT
	FOREIGN KEY
	*CURRENT_TIMESTAMP()
	
	PRIMARY KEY (ja tabulai tas ir, tad pirms pievienošanas sākuma ir japilda DROP PRIMARY KEY)
1.0.14. tabulas struktura modificēšana - ALTER TABLE TABLE_NAME ADD COLUMN_NAME DATATYPE ...;
* modifificēšana -pielikt lauku, izmainīt lauka īpašības, dzēst lauku
** ALETR TABLE TABLE_NAME MODIFY COLUMN_NAME DATATYPE ...;
***ALTER TABLE TABLE_NAME ALTER COLUMN COLUMN_NAME DATATYPE ...;
1.0.15. ierakstu pievienošana - INSERT TABLE_NAME (COLUMN_NAME) VALUES (VALUE);
1.0.16. ierakstu rediģēšana - UPDATE TABLE_NAME SET SOLUMN_NAME=VALUE WHERE COLUMN_NAME=VALUE;
1.0.17. ierakstu dzēšana - DELETE FROM TABLE_NAME WHERE COLUMN_NAME=VALUE;
